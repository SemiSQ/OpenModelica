SimForge is an open-source Graphical Editor by Politecnico di Milano.
-----------------------
 https://trac.ws.dei.polimi.it/simforge/


Installing
-----------------------
  To install just unpack the .zip from:
   OpenModelica1.5.0\SimForge\
  anywhere and then double click on SimForge-0.8.4.1.jar

  Before you can use it with OpenModelica you will need to set it up
    via Tools->Settings:
   OPENMODELICAHOME    => C:\OpenModelica1.5.0\
   OPENMODELICALIBRARY => C:\OpenModelica1.5.0\ModelicaLibrary
  Then Save Paths.

  Make sure you have Java installed as is needed to run SimForge.
  That's it. Enjoy!
