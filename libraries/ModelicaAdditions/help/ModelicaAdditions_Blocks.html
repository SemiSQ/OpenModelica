ModelicaAdditions.Blocks

Input/output block sublibrary

Information




Package ModelicaAdditions.Blocks contains
input/output blocks:




   Continuous   Continuous control blocks
   Discrete     Discrete control blocks
   Logical      Components operating on Boolean signals
   Multiplexer  Extract, combine and split signal connectors of type Real
   Nonlinear    Fixed and varying delays.
